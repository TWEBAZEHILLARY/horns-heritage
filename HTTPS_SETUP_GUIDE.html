Keep reading for setup ↓

3️⃣Production Dashboard — Enable HTTPS

If Hosting on AWS / EC2

  1. Install Certbot (Let's Encrypt): 
    sudo apt-get update
sudo apt-get install certbot python3-certbot-nginx
# or for Apache:
sudo apt-get install certbot python3-certbot-apache
  2. Get SSL certificate: 
    sudo certbot certonly --standalone -d your-domain.com
  3. Configure your server to use the certificate:
    • For Nginx: Point to /etc/letsencrypt/live/your-domain.com/
    • For Apache: Enable mod_ssl and point to certificate
  4. Restart server: 
    sudo systemctl restart nginx
# or
sudo systemctl restart apache2

If Hosting on Heroku

  1. Heroku provides automatic HTTPS on all apps
  2. Just deploy: git push heroku main
  3. Access via: https://your-app.herokuapp.com

If Hosting on DigitalOcean / Linode

  1. Use Certbot (same as AWS above)
  2. Or enable their managed SSL service

4️⃣Update Your Code for HTTPS

If you're making API calls between sites, update URLs to HTTPS:

In hh-api.js (or wherever you make API calls):

// OLD (HTTP)
const API_URL = 'http://your-server.com/api';

// NEW (HTTPS)
const API_URL = 'https://your-server.com/api';

In dashboard files (Sales Dashboard.html, admin-news-dashboard.html):

// OLD (HTTP)
fetch('http://your-server.com/api/cattle', ...)

// NEW (HTTPS)
fetch('https://your-server.com/api/cattle', ...)

Also check for hardcoded URLs in:

  • hh-api.js
  • app.js
  • dashboard-*.js files
  • Any <script src="..."> tags in HTML
