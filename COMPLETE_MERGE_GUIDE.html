Step 1: Get Cloudflare Credentials

AGet Your Account ID

  1. Go to https://dash.cloudflare.com/
  2. Click your profile (top right)
  3. Select "My Profile"
  4. Click "API Tokens"
  5. Copy your Account ID

BCreate API Token

  1. Still in API Tokens page
  2. Click "Create Token"
  3. Select "Get started" next to "Create Custom Token"
  4. Set permissions:
    • Account > Cloudflare Workers > Edit
    • Account > Cloudflare Workers Scripts > Edit
  5. Select "All accounts"
  6. Click "Create Token"
  7. Copy the token immediately (only shown once)

Step 2 & 3: Clone Repo & Copy Files

Run These Commands

# Clone your repository git clone https://github.com/TWEBAZEHILLARY/horns-heritage.git cd horns-heritage # Copy files from this workspace # (Manually copy these files/folders to the repo directory) # - wrangler.toml # - package.json # - .gitignore # - src/index.js # - .github/workflows/deploy.yml # - public/* (everything in public folder) # - assets/* (everything in assets folder) # - _ds/* (everything in _ds folder)
💡 Tip: The easiest way is to copy files using your file manager or drag-and-drop into the cloned directory.

Step 4: Push to GitHub

# Stage all changes git add . # Commit the merge git commit -m "Merge: Add deployment configuration and move files to root - Move all project files to repository root - Add wrangler.toml for Cloudflare Workers - Add GitHub Actions CI/CD workflow - Add comprehensive deployment documentation - All static files organized in public/ folder" # Push to GitHub git push origin master
✓ Success: Your files are now on GitHub at the root level!

Step 5: GitHub Secrets & Testing

Add GitHub Secrets

  1. Go to https://github.com/TWEBAZEHILLARY/horns-heritage/settings/secrets/actions
  2. Click "New repository secret"
  3. Create secret 1:
    • Name: CLOUDFLARE_API_TOKEN
    • Value: Paste your API token
    • Click "Add secret"
  4. Create secret 2:
    • Name: CLOUDFLARE_ACCOUNT_ID
    • Value: Paste your Account ID
    • Click "Add secret"

Test Deployment

  1. Make a small change (e.g., add a comment to index.html)
  2. Run:
    git add . git commit -m "Test: Verify CI/CD pipeline" git push origin master
  3. Go to https://github.com/TWEBAZEHILLARY/horns-heritage/actions
  4. Watch "Deploy to Cloudflare Workers" workflow run
  5. Once ✅ green, visit https://horns-heritage.workers.dev/
  6. Your changes are LIVE! 🎉
