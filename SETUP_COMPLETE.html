📋 Setup Instructions (4 Steps)

Step 1: Get Cloudflare Credentials

Go to https://dash.cloudflare.com/ and copy your Account ID and create an API token with Worker permissions.

Step 2: Merge with GitHub

Clone your repo, copy files from workspace, and push to GitHub. Move everything from project/ folder to root.

Step 3: Add GitHub Secrets

Go to repo Settings > Secrets and add CLOUDFLARE_API_TOKEN and CLOUDFLARE_ACCOUNT_ID.

Step 4: Test

Make a small change and push. Watch Actions tab. Once green ✅, your site is live!
