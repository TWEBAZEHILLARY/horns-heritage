⏱️ Timeline

Step 1: Get Cloudflare Credentials

Retrieve your Account ID and create an API Token from Cloudflare dashboard.

Step 2: Add GitHub Secrets

Store credentials securely in your GitHub repository settings.

Step 3: Push Workflow File

Add the GitHub Actions workflow file to your repository.

Step 4: Test & Done!

Make a change and push to test automatic deployment.

🔐 Step 1: Get API Token

From Cloudflare Dashboard:

  1. Go to https://dash.cloudflare.com/
  2. Click your profile (top right)
  3. Select "My Profile"
  4. Click "API Tokens" (left menu)
  5. Click "Create Token"
  6. Select "Get started" next to "Create Custom Token"
  7. Set permissions:
    • Account > Cloudflare Workers > Edit
    • Account > Cloudflare Workers Scripts > Edit
  8. Select "All accounts"
  9. Click "Create Token"
  10. Copy the token! (You only see it once)

📍 Get Account ID

From Cloudflare Dashboard:

  1. Go to https://dash.cloudflare.com/
  2. Click your profile (top right)
  3. Select "My Profile"
  4. Click "API Tokens"
  5. Look for "Account ID" on the page
  6. Copy your Account ID

🔑 Step 2: Add GitHub Secrets

Configure Secret 1: API Token

  1. Go to https://github.com/TWEBAZEHILLARY/horns-heritage
  2. Click Settings (repo settings, not profile)
  3. Left menu: Secrets and variables > Actions
  4. Click New repository secret
  5. Name: CLOUDFLARE_API_TOKEN
  6. Value: Paste the API token you copied
  7. Click Add secret

Configure Secret 2: Account ID

  1. Click New repository secret
  2. Name: CLOUDFLARE_ACCOUNT_ID
  3. Value: Paste your Account ID
  4. Click Add secret

📤 Step 3: Push Files to GitHub

Run these commands to push the workflow file:

git add .github/workflows/deploy.yml git commit -m "Add: GitHub Actions CI/CD workflow" git push origin master

✅ Step 4: Test Deployment

Make a Small Change:

Edit any file (e.g., update a comment in index.html)

git add . git commit -m "Test: Verify CI/CD pipeline" git push origin master

Monitor Deployment:

  1. Go to your GitHub repo
  2. Click the Actions tab
  3. You'll see "Deploy to Cloudflare Workers" workflow
  4. Wait for it to complete (2-3 minutes)
  5. Once ✅ green, your site is live!
🎉 Once Setup: Just git push and your site automatically deploys!

🚀 Ready to Setup?

Follow the 4 steps above. Total time: ~10 minutes

After setup, every push to GitHub automatically deploys to Cloudflare Workers!
