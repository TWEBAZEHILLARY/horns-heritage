# 🎯 ADDING ROUTES - EXACT STEPS

You're in the right place! Here's exactly what to do:

---

## WHERE YOU ARE NOW

You should see a page that looks like:
```
Workers & Pages > horns-heritage > Routes
```

And there might be a button that says **Add route** or **Create route**

---

## STEP 3A: ADD FIRST ROUTE (Main Website)

**Click the "Add route" button**

A dialog box will pop up with fields to fill in:

### Field 1: Route
Type exactly this:
```
hornsandheritage.com/*
```

### Field 2: Worker
- Click the dropdown that says "Select a worker"
- Choose: `horns-heritage`

### Field 3: Zone
- Click the dropdown that says "Select a zone"
- Choose: `hornsandheritage.com`

**Then click SAVE**

You should see this appear in your routes list:
```
✓ hornsandheritage.com/* → horns-heritage
```

---

## STEP 3B: ADD SECOND ROUTE (Dashboard Subdomain)

**Click "Add route" again**

Another dialog box pops up:

### Field 1: Route
Type exactly this:
```
dashboard.hornsandheritage.com/*
```

### Field 2: Worker
- Click the dropdown
- Choose: `horns-heritage`

### Field 3: Zone
- Click the dropdown
- Choose: `hornsandheritage.com`

**Then click SAVE**

You should now see BOTH routes in your list:
```
✓ hornsandheritage.com/*          → horns-heritage
✓ dashboard.hornsandheritage.com/* → horns-heritage
```

---

## ✅ YOU'RE DONE WITH ROUTES!

Once both routes appear with green checkmarks ✓, move to **STEP 4: Deploy the Code**
